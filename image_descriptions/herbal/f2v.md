# f2v — herbal 섹션

## 1. 페이지 구성

- **레이아웃**: 거대한 단일 잎이 페이지 중앙 차지. 꽃은 페이지 상단 우측에 작은 크기로 분리.
- **텍스트 블록**: 시각적 3개 (상단 좌측, 하단 좌·우)
- **이미지 영역 비율**: ~75%
- **특이 구성**: 일반 식물 도해와 달리 *잎 한 장*이 페이지 중심.

## 2. 색상 팔레트

| 색 | 사용 영역 |
|----|-----------|
| 갈색 잉크 | 윤곽선, 텍스트 |
| 녹색 (진하고 균일) | 거대한 단일 잎 — 단색 채색 |
| 황토/연갈 | 줄기 |
| 베이지/연크림 | 페이지 하단의 작은 원형 점들의 띠 |

## 3. 객체 인벤토리

- **단일 식물**:
  - **잎**: 1장. 둥근 심장형(orbicular/cordate), 페이지의 거의 절반 차지. 진한 녹색 단색.
  - **꽃**: 1개. 페이지 우상단 작은 크기. 흰/크림색 종형, 5-6개의 결각.
  - **줄기**: 가늘고 긴 황토색 라인, 잎과 꽃을 연결. 페이지 하단까지 직선으로 내려감.
  - **뿌리**: 명시적 뿌리 없음. 대신 페이지 하단에 *작은 원형 점들의 수평 띠* (~30개의 작은 동그라미가 줄지어 있음).

## 6. 특이 관찰

- **하단의 원형 띠**: 다른 식물 페이지에서 보기 드문 모티프. 자갈/씨앗/뿌리혹/지표면 등으로 해석 가능 (해석 보류).
- **잎 1장 강조**: 잎-중심 구도. f87 등 다른 잎-중심 페이지와 비교 가치.
- **줄기와 꽃의 비례 부조화**: 거대 잎에 비해 꽃이 매우 작음.

# f111r — recipes

## 1. 페이지 구성

텍스트 전용 페이지. 좌측 마진에 단락별 별 1개. 약 16개 단락 + 별.

## 2. 색상 팔레트

| 색 | 사용 |
|----|------|
| 갈색 잉크 | 본문 |
| 황토 | 별 일부 |
| 적색/주홍 | 일부 별 (드물게) |

## 3. 객체 인벤토리

- **본문 단락** ~16개. 각 단락 시작에 좌측 마진 별 1개.
- **별 모양**: 7-8각, 작음. 일부는 빨강·꼬리(혜성형) 가능.

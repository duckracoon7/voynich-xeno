# f3v — herbal 섹션

## 1. 페이지 구성

- 식물 페이지 우측 수직, 텍스트 좌상+좌중 2 블록.
- 이미지 영역 ~55%.

## 2. 색상 팔레트

| 색 | 사용 영역 |
|----|-----------|
| 갈색 잉크 | 윤곽, 텍스트 |
| 녹색 | 잎 |
| **청색 (선명)** | 정점 콘 2개 |
| 베이지/크림 | 콘 하단 |
| 적갈/주홍 | 뿌리부 수평 체인 |

## 3. 객체 인벤토리

- **꽃 (정점)**: 2개. 청색 솔방울/콘, 하단 흰/크림 비늘. 큰 1 + 작은 1.
- **잎** ~10: 길쭉 가시 모양(palmately-lobed/spiky), 녹색 단색, 좌우 짝.
- **줄기**: 직선 수직, 가는 갈색.
- **뿌리부**: 일반 뿌리 없음. **횡 누운 적갈색 곡선 막대(체인)**, 표면에 흰색 결절(고리/마디).

## 6. 특이 관찰

- 체인/사슬 뿌리 — 매우 드문 모티프. 신화·혼합 도해 가능성.
- 청색 안료 등장 페이지 (안료 사용 통계 표지).
- 같은 quire 내에서도 잎 형상 다양성 매우 큼.

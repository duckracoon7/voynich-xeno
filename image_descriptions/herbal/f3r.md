# f3r — herbal 섹션

## 1. 페이지 구성

- 식물 페이지 우측, 텍스트 좌측 (좌→우 비대칭).
- 텍스트 블록 3 (시각).
- 이미지 영역 ~50%.

## 2. 색상 팔레트

| 색 | 사용 영역 |
|----|-----------|
| 갈색 잉크 | 윤곽, 텍스트 |
| 녹색 | 잎 본체 |
| **적색/주홍** | 잎의 줄무늬 (강렬) |
| 적갈/오렌지 | 뿌리 |
| 청록 (희미) | 후면 비침 |

## 3. 객체 인벤토리

- **꽃 (정점)**: 3-4개 적색 가시/촛불 모양 돌출.
- **잎** ~10: 수평 적층, 길쭉한 잎이 줄기 좌우 한 쌍씩. **녹색+적색 줄무늬**.
- **줄기**: 직선 수직, 갈색.
- **뿌리**: 손가락 모양 7-8개 갈래, 발사형.

## 6. 특이 관찰

- 녹+적 줄무늬 잎 — 매우 독특.
- 손가락형 분기 뿌리 → f1v의 둔덕형과 대조.
- 정점 적색 가시가 꽃이라기보다 가시.

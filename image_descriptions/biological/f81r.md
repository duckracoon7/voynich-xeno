# f81r — biological (목욕/생물 섹션)

## 1. 페이지 구성

- 본문 텍스트가 페이지 대부분.
- 마진과 텍스트 사이에 누드 여성 인물 (nymph) 다수.
- 일부 페이지: 녹색 풀/욕조, 파이프/관, 용기.

## 2. 색상 팔레트

| 색 | 사용 |
|----|------|
| 갈색 잉크 | 본문, 윤곽 |
| **녹색** | 풀/욕조 물 (특징적) |
| 청색 (드물게) | 일부 물줄기·용기 |
| 적색 | 일부 인물 의복/액센트 |
| 황토 | 인물 머리/피부 |

## 3. 객체 인벤토리

- **인물 (nymph)**: ~13명. 누드, 일부 모자/관 착용.
- **풀/욕조**: 2개.
- **파이프/관/연결구**: 0개.
- **자세 다양**: 서기/앉기/누워있기/물에 잠기기.

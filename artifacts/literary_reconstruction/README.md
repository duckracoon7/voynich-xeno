# 문학적 재구성 (Literary Reconstruction)

## ⚠️ 매우 중요한 면책 — 반드시 읽을 것

**이 폴더의 글은 보이니치 문서의 *번역*이 아닙니다.**

이것은:
- ✗ 보이니치 텍스트의 의미 해독이 아님
- ✗ 어떤 학술적 발견의 주장이 아님
- ✓ 우리 분석 결과(시각 anchor + 섹션 분류)를 토대로 *그 시대 그 장르의 책*이 어떻게 읽혔을지를 *상상*한 글
- ✓ 명시적으로 표시된 *추측 산문* (speculative prose)

### 왜 이런 글을 만드는가

지금까지 우리는:
1. 226 폴리오를 EVA로 정규화 (Phase 1)
2. 통계적으로 자연언어 구조 확인 (Phase 2: h₂≈2)
3. 토큰 임베딩 학습 + 검증 (Phase 3-4)
4. 토큰-시각특성 anchor 추출 (Phase 7)
5. 통계적 유의성 입증 (Phase 9: 50/50 p<0.01)
6. 토큰별 추정 라벨링 산출 (Phase 8, 10)

하지만 사용자가 *읽힐 만한 산문*을 원함.
Phase 8/10의 "사람✦ 풀★ 관☆"는 신뢰성은 높으나 가독성 0.

이 폴더는 Phase 8/10의 anchor 신호 + 페이지 시각 사실 + 중세 그 장르 컨벤션을
**자유롭게 결합**한 한국어 산문. **결정적 의미가 아니라 분위기**를 전달.

### Cheshire 2019 함정과 어떻게 구별되는가

Cheshire는 자신의 산문을 *해독 결과*로 발표 → 비판받음.

우리는:
1. 산문이 *문학적 재구성*임을 매 페이지 상단에 굵게 표시
2. 각 페이지 하단에 *anchor 토큰 검증 표* 첨부 — 어떤 EVA 토큰의 어떤 anchor가
   어떤 한국어 단어로 변환되었는지 추적 가능
3. 산문 자체에 [확실: 시각 사실] / *추정: anchor* / **상상: 컨벤션 보강** 표시
4. *해독 주장 절대 하지 않음*

### 활용 방법

- **읽기**: 페이지 분위기·내용 직관적 파악
- **검증**: 하단 anchor 표로 어떤 부분이 데이터에서 왔는지 추적
- **반증**: 더 정확한 해독이 나오면 즉시 폐기 가능 (이는 가설이지 결론이 아님)

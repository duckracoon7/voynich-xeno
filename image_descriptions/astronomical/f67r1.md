# f67r1 — astronomical (foldout 좌측)

## 1. 페이지 구성

- 폴드아웃 좌측 패널. 거대 원형 다이어그램 1개가 페이지 대부분 차지.
- 텍스트: 다이어그램 상단 라인 4-5줄 + 다이어그램 둘레 원형 텍스트 2개 동심.

## 2. 색상 팔레트

| 색 | 사용 영역 |
|----|-----------|
| 갈색 잉크 | 텍스트, 윤곽 |
| 청색 (선명) | 16개 광선 중 절반 |
| 적색/주홍 | 16개 광선의 나머지 절반 (교차) |
| 황토 | 중앙 얼굴 |
| 노랑 (희미) | 외곽 별 점들 |

## 3. 객체 인벤토리

- **중앙 객체**: 사람 얼굴 (sun face), 둥근 황토색.
- **광선**: 16개의 길쭉한 삼각 광선이 *청-적-청-적* 교차 배열.
- **외곽**: 광선 끝 부근에 작은 별 모양 점들 다수 (~30개 추정).
- **둘레 텍스트**: 2개의 동심 원호를 따라 흐름.
- **상단 본문**: 4-5라인 직선 텍스트.

# f70v1 — zodiac

## 1. 페이지 구성

- 큰 원형 황도궁 휠. 페이지 중앙.
- 중심 엠블럼 + 동심 2 링 + 외곽 원형 텍스트.

## 2. 색상 팔레트

| 색 | 사용 |
|----|------|
| 갈색 잉크 | 윤곽, 텍스트 |
| 청록 (희미) | 중심 엠블럼 주변 |
| 황토 | 별 일부 |

## 3. 객체 인벤토리

- **중심 엠블럼**: 두 마리 물고기 (Pisces 도해 — 동정 보류). 한 마리는 위쪽, 한 마리는 아래쪽, 서로 반대 방향.
- **동심 링 1 (안쪽)**: ~10명의 나신 인물, 각자 8각 별을 들고 있음.
- **동심 링 2 (바깥쪽)**: ~20명의 인물, 각자 별 1개. 각 인물 옆 짧은 EVA 라벨.
- **외곽 텍스트**: 둘레 원형 한 줄.

# f2r — herbal 섹션

## 1. 페이지 구성

- 식물 중앙 수직축, 텍스트 상단 좌·중 + 하단 좌·우 분산.
- 텍스트 블록 4 (시각).
- 이미지 영역 ~70%. 텍스트가 식물 회피 (이미지 선행).

## 2. 색상 팔레트

| 색 | 사용 영역 |
|----|-----------|
| 갈색 잉크 | 윤곽, 텍스트 |
| 녹색 (mid) | 잎 (palmately-lobed) |
| 황토/베이지 | 꽃 (4개 술 모양) |
| 적갈/주홍 | 줄기, 뿌리 |
| 청록 (희미) | 페이지 중앙 얼룩/후광 |

## 3. 객체 인벤토리

- **꽃** 4: 줄기 끝, 황토색 술/브러시 형태, 일부 녹색 받침.
- **잎** ~10: 손가락 모양(palmate-lobed), 깊게 분열.
- **줄기**: 적갈색 채색, 4-5개 측지.
- **뿌리**: 작은 곡선형 적갈색 매스.

## 6. 특이 관찰

- 줄기 자체가 적갈색 채색 (드문 사례).
- 꽃 갯수 정확히 4 → 페이지 간 카운트 통계 자료.
- 잎이 깊은 분열형, 단순 타원과 구별.

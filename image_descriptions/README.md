# 보이니치 문서 페이지별 이미지 설명 (Image Descriptions)

> 본 디렉터리는 연구 계획서 Phase 5(이미지-텍스트 매핑)의 **입력 데이터**다.
> 각 폴리오의 이미지를 *관찰 가능한 형태적 특성*으로만 기술한다.

---

## 작성 원칙 (필독)

연구 계획 P4에 따라 다음을 **엄격히** 준수한다:

### ✅ 해야 할 것
- **관찰**: 색상, 형태, 구성 요소의 개수, 위치, 크기, 배열
- **정량화**: "잎 7장", "별 12개", "4단으로 분기" 처럼 **셀 수 있는 것은 센다**
- **상대적 위치**: "페이지 상단 1/3", "중앙 좌측" 등
- **텍스트와의 공간 관계**: 텍스트 블록의 개수, 이미지 주변 배치 (위/아래/감쌈)
- **라벨(label)의 존재**: 그림 가까이 별도로 짧게 쓰인 단어가 있는지 표시

### ❌ 하지 말아야 할 것
- **식별/동정 금지**: "이것은 해바라기다", "이 별자리는 양자리다" 같은 *해석*
- **기능 추정 금지**: "약초의 효능을 나타내는 그림"
- **언어적 의미 부여 금지**: 텍스트의 뜻을 추측하지 않음
- **다른 문헌과의 매핑 금지**: 이 단계에서는 *순수 관찰*만

> **이유**: Cheshire 2019의 함정(이미지에서 의미를 끼워 맞추는 순환 논증)을 회피.
> 이미지 *동정*은 ground truth로 쓰지 않으며, 형태 벡터화의 입력으로만 사용된다.

## 디렉터리 구조

```
image_descriptions/
├── README.md                       # 본 파일
├── template.md                     # 표준 작성 템플릿
├── section_inventory.md            # 섹션별 폴리오 인벤토리
├── herbal/                         # 식물 섹션 (f1r 제외 q1, q2-q7, f57r, f65, f66v, f87, f90, q17)
│   ├── f1r.md
│   ├── f2v.md
│   └── ...
├── astronomical/                   # 천문 (f67, f68 일부)
├── cosmological/                   # 우주 (f67v2, f68v3, f69-f70, f85r2, f86v3-4)
├── zodiac/                         # 별자리 (f70v1-f70v2, f71-f73)
├── biological/                     # 생물/목욕 (Quire 13 전체)
├── pharmaceutical/                 # 약초 (f88-f89, Quire 19)
└── recipes/                        # 요리법/별 (f58, Quire 20 - f116v 제외)
```

## 작업 진행 방식

1. **Phase 0 (현재)**: 템플릿 + 섹션별 견본 1–2개씩 작성
2. **Phase 1**: 견본 검토 후 식물 섹션(가장 단순) 전체 확장
3. **Phase 2**: 천문/별자리 섹션 (라벨 단어가 풍부 → 매핑 후보)
4. **Phase 3**: 생물/약초/우주 섹션 (복잡)
5. **Phase 4**: 요리법 섹션 (별 갯수만 기록)

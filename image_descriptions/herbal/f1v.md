## 2. 색상 팔레트

| 색 | 사용된 영역 |
|----|-------------|
| 갈색/적갈색 (잉크) | 줄기 윤곽, 잎 윤곽, 뿌리 해칭, 텍스트 |
| 녹색 (sage/medium green) | 잎 일부 (약 절반) |
| 황토/연주황 (ochre) | 잎의 나머지 절반 |
| 어두운 갈색/흑갈색 | 꽃 상단의 돔 캡 |

**색의 비율 인상**: 잎은 녹색과 황토색이 거의 1:1로 교차, 줄기·뿌리는 단색 잉크선만, 꽃은 짙은 갈색(상단)과 흰색(하단) 대비.

## 3. 이미지 객체 인벤토리

### 객체 1 — 단일 식물 (전체 페이지의 주된 객체)

- **위치**: 페이지 중앙 약간 좌측, 상단부터 하단까지 수직 배치
- **크기 (상대)**: 페이지 가로의 약 50%, 세로의 약 65%
- **줄기/축**: 직선에 가까운 단일 수직 줄기, 양측으로 대칭에 가까운 5–6 단의 측지 분기 (계층적 분기)
- **잎 모양 구조**:
  - 형태: 란셰올레이트(lanceolate, 길쭉한 타원-피침형), 끝이 뾰족함, 가장자리 매끈
  - 갯수: 약 28–30 (좌우 측지 군집에 5–6장씩 묶여 있음)
  - 색 분포: 녹색 ↔ 황토색이 잎 단위로 교차 (개별 잎 안에서 그라데이션 아님)
- **꽃 모양 구조**:
  - 갯수: 1 (꼭대기 단 하나)
  - 구조: 짙은 갈색 돔/스케일 캡(약 2/3) + 그 아래 흰색 둥근 화관/포(envelope-like, 4–5 결각)
  - 위치: 줄기 정점, 좌우 대칭
- **뿌리 모양 구조**:
  - 형태: 양 옆으로 넓게 퍼진 두 개의 둥근 둔덕 모양 텍스처, 수평 해칭(빗금)으로 빽빽이 채워짐. 식물의 캐노피보다 더 넓은 폭.
  - 일반적인 "뿌리(섬유근/덩이)"와 다름 — 토양/이끼처럼 보이는 텍스처 매스
- **라벨**: 없음 (객체 옆에 별도 단어 표기 없음)

## 6. 특이 관찰

- **꽃 구조의 비전형성**: 짙은 돔 + 흰색 화관 결합은 일반 화훼 도해와 다름. 솔방울/엉겅퀴/과실의 변형 도해 가능성 (해석 보류).
- **뿌리부의 스타일**: 뿌리라기보다 **두 개의 둔덕**으로 보이는 텍스처. 다른 식물 페이지와 비교 필요.
- **잎의 색상 교차**: 녹색/황토 교차 패턴이 자연 식물과 다름. 색상이 형태 식별의 단서가 아닐 수 있음(장식적 변화 vs 의미적 차이 — 후속 페이지 비교로 검증).
- **줄기와 텍스트의 관계**: 본문이 줄기를 *피해* 흐름 → 그림이 먼저 그려졌고 텍스트가 나중. (Bowern & Lindemann의 일반 관찰과 일치)
- **양피지 손상**: 좌측 가장자리에 뚜렷한 자국(밴드/봉합 흔적), 페이지 전반에 작은 검은 얼룩 다수.

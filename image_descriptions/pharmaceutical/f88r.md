# f88r — pharmaceutical

## 1. 페이지 구성

- 좌측: 3개의 정교한 단계형 용기 (위→아래 적층).
- 우측 상단: 식물 단편 6개 (수평 줄).
- 중간/하단에 각각 텍스트 단락.
- 3행(row) 구조: [용기 + 단편 + 본문] × 3.

## 2. 색상 팔레트

| 색 | 사용 |
|----|------|
| 갈색 잉크 | 본문, 윤곽 |
| 적색/주홍 | 용기 줄무늬, 일부 단편 (붉은 뿌리) |
| 녹색/청록 | 용기 받침/단(stem), 단편 잎/뿌리 |
| 황토 | 일부 단편, 식물 줄기 |

## 3. 객체 인벤토리

- **용기 1 (상단)**: 단계형 적-녹 줄무늬, 정교한 다단 화병형.
- **용기 2 (중단)**: 더 큰 단계형, 받침 부분 발달.
- **용기 3 (하단)**: 작고 길쭉한 첨두형 (탑 모양).
- **식물 단편 (행 1)**: 6개. 잎+뿌리 혼합 (`osal`, `orald`, `oldar`, `otoky`, `otaly` + 1 unnamed?).
- **식물 단편 (행 2)**: 5개 (`oram`, `dary`, `okol`, `sorory`, `otyda?`).
- **식물 단편 (행 3)**: 2개 (`otoram`, `ofaldo`).
